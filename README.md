## 🛠️ 技术栈

- **框架**: Flutter 3.0+
- **语言**: Dart 3.0+
- **架构**: MVVM + 模块化
- **状态管理**: Provider + Bloc
- **路由**: Flutter Modular
- **网络**: Dio
- **存储**: Hive + SharedPreferences

## 📖 使用指南

### 在线浏览
- 使用左侧导航栏浏览不同模块
- 使用右上角搜索功能快速查找内容
- 点击主题切换按钮切换深色/浅色模式

### 本地部署
```bash
# 安装 docsify-cli
npm install -g docsify-cli

# 在项目目录启动本地服务
docsify serve

# 访问 http://localhost:3000
```
